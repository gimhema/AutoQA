# TheWeapons

무기 카드를 소켓에 배치해 상대의 HP를 먼저 소모시키는 2인 턴제 카드 게임.
매 턴 양측의 카드가 동시에 공개되어 효과가 적용된다.

---

## 세팅

게임 시작 전 두 플레이어가 합의해 결정한다.

| 항목 | 설명 | 예시 |
|------|------|------|
| **소켓 수** | 한 턴에 낼 수 있는 카드 슬롯 수 | 3 |
| **초기 HP** | 각 플레이어의 시작 체력 | 10 |
| **핸드 크기** | 각 플레이어가 처음 받는 카드 수 | 15 |

카드 구성 비율도 세팅 항목이다. 기본 구성 예시:

```
검 5장 / 방패 5장 / 창 5장  (총 15장)
```

---

## 카드

| 카드 | 효과 |
|------|------|
| **검** | 상대 HP를 1 감소시킨다. 상대의 방패에 막힌다. |
| **방패** | 이번 턴 상대의 검 1개를 차단한다. 창에는 무력하다. |
| **창** | 방패를 무시하고 상대 HP를 1 감소시킨다. |

모든 카드는 **소모형**이다. 소켓에 배치하면 효과 발동 후 손에서 제거된다.

---

## 카드 상호작용

### 1대1 상성

```
창  →  방패를 무시
방패  →  검을 차단
검  →  빈 소켓(무방비)에 유효
```

### 상황별 결과표

| 내 카드 | 상대 카드 | 결과 |
|---------|-----------|------|
| 검 | 없음(빈 소켓) | 상대 HP -1 |
| 검 | 검 | 둘 다 HP -1 |
| 검 | 방패 | 방패가 검을 차단 (둘 다 소모) |
| 검 | 창 | 내 HP -1, 상대 HP -1 |
| 방패 | 검 | 방패가 검을 차단 (둘 다 소모) |
| 방패 | 방패 | 아무 효과 없음 (둘 다 소모) |
| 방패 | 창 | 방패 무력화, 내 HP -1 |
| 창 | 없음(빈 소켓) | 상대 HP -1 |
| 창 | 검 | 둘 다 HP -1 |
| 창 | 방패 | 방패 무시, 상대 HP -1 |
| 창 | 창 | 둘 다 HP -1 |

### 소켓이 여러 개일 때

소켓별로 독립적으로 처리한다. 방패 1장은 같은 턴에 낸 검 1개만 차단한다.

**예시** (소켓 3개):

```
나:   검  방패  검
상대: 창  검    없음

소켓1: 검 vs 창  → 내 HP -1, 상대 HP -1
소켓2: 방패 vs 검  → 방패가 검 차단 (둘 다 소모)
소켓3: 검 vs 없음  → 상대 HP -1
```

## 턴 진행

```
1. 양측이 동시에 각 소켓에 카드를 배치하거나 비워둔다.
2. 턴 개시: 양측 카드가 동시에 공개된다.
3. 소켓 번호 순서대로 각 소켓의 카드 효과를 적용한다.
4. 사용된 카드는 모두 손에서 제거된다.
5. 다음 턴으로 이동한다.
```

---

## 승리 / 패배 조건

- **승리**: 상대 HP를 0 이하로 만들면 즉시 승리한다.
- **카드 고갈**: 양측 모두 손에 카드가 없어지면 게임 종료.
  - HP가 높은 쪽이 승리한다.
  - HP가 동일하면 무승부.
- **소켓 비우기**: 소켓 일부 또는 전부를 비워두는 것은 합법이다. 카드를 아끼는 전략으로 활용할 수 있다.

---

## 전략 요점

- **방패 비축**은 창에 취약하다. 상대가 창을 많이 가진 상황이라면 방패는 손해.
- **창 남발**은 방패가 없는 상대에게 낭비다. 상대 패를 읽어 검과 창을 구분해 써야 한다.
- **소켓을 비우는 패스**는 카드 고갈을 늦추지만, 상대가 적극적으로 공격하면 HP 차이로 불리해진다.
- **카드 고갈이 가까울수록** 남은 HP 차이가 승패를 결정하므로, 후반부엔 공격 비중을 높이는 것이 유리하다.

## Ouroboros 에이전트 연동 (예정)

> 현재 게임 구현 전 단계. 아래는 예정된 연동 방식이다.

**Step 1** — 터미널 A: 게임 실행 (에이전트 접속 대기)
```bash
cd Games/TheWeapons
cargo run --release -- ai --ouroboros-port 9000
```

**Step 2** — 터미널 B: 에이전트 접속
```bash
cd Ouroboros
cargo run --release -- 127.0.0.1:9000 "상대의 HP를 먼저 소진시켜라" \
  --action-space dynamic \
  --rulebook ../Games/TheWeapons/Rule/RULEBOOK.md
```
